Datatypes in C#

In c# ,when we declare the variable we also need to specify the data-type for that variable like integer, string, float, decimal etc.

There are two kind of data-types in c#

1.      Values Types

2.      Reference types


Value types are those variables which directly contain values like integer, float, char, double, boolean. Reference type variable store reference of their data, In reference type can have its own copy of data.

Data-type syntax in c#
Flowing is the syntax for declaration of c# datatype:

 

   (Data-type)   [variable name];

   (Data-type)   [variable name] = [Value];

 

 

 

As we see the syntax for data types first we put the data type (like integer, float, boolean) and then put the name of the variable.

 

Value-Types

The values type directly contain Data. Some examples of value types are integer, float, boolean etc.

The value type can be further divided by two categories

1.      Predefined:  Integer, float, char, boolean etc.

2.      User-Define : Struct, Enumerable, Nullable etc.

Reference Types

The Reference types contain the reference of the variable not the actual data.In other word, it points to the location of the memory where actual data. Some of the examples of reference types are class, Array, interface, Delegate etc.

1.Class:  The class is a blueprint of an object. It is a template for the object , it contains methods, variables and constructors.

Let see a simple example of c# class

 {
