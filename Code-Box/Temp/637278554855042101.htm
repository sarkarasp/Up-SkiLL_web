C# Programming Structure

To understand the programming structure of C#, first we need to create a console application in visual studio IDE.

Follows the below steps to create our first c# program in Visual Studio.

 

1.      Open the Visual Studio Editor click on Create a new project under the Get Started section.

2.      In the Create New Project section, Select Console App (.NET Framework) and click on Next button.

 

3.      In Configure your new project section, enter project name, Location, Solution name and click on Create button.
We can place solutions and projects in the same directory by applying a checkbox and also choose Framework Version.
