C# Introduction
C# is created in Microsoft in late 1990 by Anders Hejlsberg. C# is a modern, general-purpose, object oriented programming language. The syntax is similar to c and c++ but many feature of those not supported by C#.
The .NET Framework defines an environment that supports the development and execution of C#. Using C# and .NET Framework we can create different types of applications like web Application, websites, desktop application, mobile application, games and many others. The .NET Framework also supports other languages like F#, Visual Basic etc.

The C# is running on a special environment called Common Language Runtime (CLR) inside .NET Framework. Apart from CLR, .NET Framework also consists of a bundle of libraries, compilers, debuggers and other development tools. C# language is mostly used to develop software components suitable for deployment in distributed environments.
