Environment Set-up
C# programming language has been build on .NET Framework to build various application like Windows, Web, web Services. To set up the Environment of C# programming languages required two components like .NET Framework and Visual studio IDE.

.NET Framework
To run C# or any other programming language, it required to installing .NET Framework on your System. If you have windows operation system, the .NET Framework is already installed o your system. The .NET Framework help us to build different type of web applications like Windows , web based applications using different programming language like C#, Visual Basic, F# etc. It Consist different two basic components like Common Language Runtime (CLR) and Framework Class Library (FCL).

Visual Studio IDE (Integrated Development Environment)
